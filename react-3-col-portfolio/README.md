# react-3-col-portfolio

Isolated React conversion of the original `3-col-portfolio` template.

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

Notes:
- This conversion keeps the original layout and content style while replacing static HTML/JS with React components.
- The project is fully isolated in this folder and does not share runtime files with other templates.
