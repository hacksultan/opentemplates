Split Index Template

Split Index is a free single-page HTML CSS portfolio template with a distinctive split-panel layout. The viewport is divided into two halves where the left side swaps between Work, About, and Contact sections, while the right side acts as a full-height image canvas that responds to user interaction. Hovering over project titles triggers smooth fade-and-scale image previews, creating an engaging experience without page reloads or scrolling.

The design follows a warm paper aesthetic with a subtle grain texture, pairing Playfair Display serif headings with DM Mono monospace for metadata and Instrument Sans for body text. Interactive elements feature black-fill wipe transitions on hover, and the contact section includes a minimal underline-style form. The entire template is contained in a single HTML file, making it easy to edit manually or customize with AI tools like ChatGPT and Claude.

Split Index is a fully responsive website template with touch-friendly tap interactions on mobile. Built with pure CSS transitions and minimal vanilla JavaScript, it is ideal for photographers, architects, designers, and visual artists looking for a free photography website template that feels cinematic rather than conventional.
