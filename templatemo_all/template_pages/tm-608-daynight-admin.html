Daynight Admin Template

DayNight Admin is a clean, flat dashboard template featuring dual theme modes - Snow Edition (light) and Carbon Edition (dark). Built with pure HTML, CSS, and vanilla JavaScript, this template requires no external dependencies for its charts and interactive elements.

The template includes 6 fully-designed pages: Dashboard with revenue charts and activity feed, Projects with a draggable Kanban board, Inbox with split-pane message view, Analytics with traffic comparison and source breakdown, Settings with organized preference panels, and a Login page. Theme preferences persist via localStorage with zero white-flash on page navigation.

Designed to be AI-customizable, you can easily modify colors, layouts, and content by uploading the files to AI tools like Claude. The responsive design adapts seamlessly from desktop to mobile with a slide-out navigation menu. Typography uses DM Sans from Google Fonts, paired with an Ice Blue accent color that works beautifully in both light and dark modes. Perfect for admin panels, SaaS dashboards, or as a starter template for your web application projects.
