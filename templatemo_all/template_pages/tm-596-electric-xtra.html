Electric Xtra Template

Electric Xtra is a dynamic and futuristic HTML landing page template designed with a striking cyberpunk aesthetic. The hero section immediately grabs attention with an animated text rotator that showcases multiple headlines using a cool glitch effect. The template is built with pure HTML, CSS, and vanilla JavaScript, ensuring it's lightweight and easy to customize without any external library dependencies.
